# AI-дайджест для новичков

Сайт-картотека с полезными материалами о нейросетях, Claude Code, промптах и первых проектах — всё простым языком для новичков.

## Возможности

- Карточки с заголовком, описанием, тегами и ссылкой
- Поиск по названию, описанию и тегам
- Фильтр по тегам (клик — фильтр, повторный клик — сброс)
- Тёмная тема с неоновыми градиентами и анимациями
- Адаптив под телефон и компьютер

## Установка и запуск

```bash
pip install flask
python app.py
```

Открой в браузере: http://127.0.0.1:5000

## Структура проекта

```
├── app.py              # Flask-сервер
├── content.json        # Данные карточек
├── templates/
│   └── index.html      # HTML-шаблон
└── static/
    ├── style.css       # Стили и анимации
    └── script.js       # Поиск и фильтрация
```

## API

- `GET /api/cards` — все карточки
- `GET /api/tags` — все теги

## Добавление карточек

Открой `content.json` и добавь новый объект в массив:

```json
{
  "id": 11,
  "title": "Название материала",
  "description": "Краткое описание",
  "url": "https://ссылка.на/материал",
  "tags": ["тег1", "тег2"]
}
```
